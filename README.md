# 🔍 Smart Port Scanner Elite (Cybersecurity Tool)

An advanced GUI-based multi-threaded port scanner built using Python to detect open ports and identify running services using banner grabbing techniques.

![Scanner UI](screenshot.png)

---

## 🚀 Features

- ⚡ Multi-threaded fast scanning  
- 🧠 Service detection (banner grabbing)  
- 🖥️ Dark-themed GUI interface  
- 📊 Real-time results display  
- 📈 Progress bar tracking  
- 🛑 Start/Stop scan control  
- 📁 JSON export support  

---

## 🧠 How It Works

- Uses **socket programming** to attempt connections to ports  
- Identifies open ports based on successful connections  
- Extracts service information using **banner grabbing**  
- Implements **multi-threading** to scan multiple ports simultaneously  

---

## 🔐 Why This Project?

Port scanning is an essential technique in cybersecurity used to identify open ports and potential vulnerabilities in a system.

This project simulates a real-world security tool that helps understand how attackers and security analysts analyze network services.

---

## 🖥️ Tech Stack

- Python  
- Socket Programming  
- Threading  
- Tkinter
